What’s new in RapidReconciler.

Newest changes appear at the top — each entry is auto-published from a merge to main. Spot something missing or want context on a fix? rrsupport@getgsi.com.

May 20, 2026

- The Help Desk Log Analyzer now reads `.out.log` files via a drop zone instead of pasting into a textarea -- multi-megabyte end-of-day logs that were slow or unresponsive to paste now load instantly. Drop a log file onto the orange zone in the RR agent card, or click to browse. - Repeated occurrences of the same finding collapse into a single row with a count and a "Show N more occurrences" expander -- so a log with hundreds of routine warnings is easy to scan instead of a wall of duplicate lines. - Unknown errors that spam the log dedupe to one row each based on their shape, ignoring timestamps and instance-specific values.

May 20, 2026

- Log Analyzer now treats two known-benign Spring Boot startup warnings as expected noise instead of flagging them: the HikariCP Derby driver SPI miss ("Registered driver was not found, trying direct instantiation") and the Hibernate "Unknown Microsoft SQL Server major version" dialect fallback. Both appear in every healthy RR Agent log and aren't actionable.

May 19, 2026

- A new 5-minute interactive self-guided demo for RapidReconciler Assist now lives at GSIRRSales/rr-assist-self-guided-tour.html. It walks through each of the four Assist surfaces -- University+, Export Analyzer, Self Help Desk, and Log Analyzer -- with a real interactive moment in each stop. - The demo is linked from the RapidReconciler Assist help portal hero ("Prefer to explore at your own pace? Take the 5-minute self-guided demo") and from the hub's Demos section.

May 19, 2026

- RapidReconciler is licensed per JD Edwards company number, not per user. The new RapidReconciler Licensing page (under RapidReconciler University+) walks through what GSI Sales needs to add a new JDE company to your entitlement. - The hub's "What's inside" panel has cleaner labels (Products / Training / Workflows / Demos / Documents) and tighter, evenly-sized item chips.

May 19, 2026

- RapidReconciler Assist is now the dedicated brand for the help portal -- a one-stop home for the knowledge base, Help Desk, Export Analyzer, and Log Analyzer. - The knowledge base is now RapidReconciler University+ -- the expanded edition with deeper coverage across Inventory, A/P, Transfers, and Administrator. - The help portal landing page is redesigned: a single prominent "Take the 90-second tour" button in the hero, and all four destinations -- University+, Export Analyzer, Help Desk, and Log Analyzer -- on one clean row.

May 19, 2026

- All 75 customer-facing and internal docs across RR University, Scenarios, Help Desk, GSI Tech, and GSI Sales now share a single header-template loader instead of duplicating the same banner HTML on every page. Visual output is identical; the loader fills in the wordmark, the doc-kind pill, and the GSI brand mark at page load. - One change to the shared template (logo, layout, accent color, anything) now propagates to every doc instead of needing 75 file edits. - update_doc_dates.py (the GitHub Action that auto-stamps each doc's last-updated date on push to main) keeps working without any changes -- the last-updated span stays inline on each page so the script can find and update it.

May 19, 2026

- New RR University reference doc: "JDE Order & WO Status Codes Reference" (UDC 00/SS for work orders, 40/AT for sales and purchase orders, plus Order Activity Rules). Searchable on NxtSts, NXTR, LSTS, WrStts, and the underlying F-tables. - Added to the University drawer and the Help Desk Browse-all drawer. - 40 docs across HelpDesk, GSIRRTech (including 16 install scenarios), and GSIRRSales now share the same doc-header template as RR University and Scenarios. Wordmark, page-kind pill, last-updated date, and GSI mark on every doc -- no more topnav-format mix.

May 19, 2026

- The four destination pages (RR University, Export Analyzer, Help Desk, Log Analyzer) now share the same welcome-banner template -- wordmark, page pill, Beta, GSI. No more divergent in-banner action buttons or stale topnav formats. "How to use this page" links live in each tool's hero now. - The Export Analyzer's GSI logo is back to the shared 96px standard (was a stale 64px override). - The "Use & analyze" and "Troubleshoot" group labels on the help cover page are now visible (were near-invisible grey on white). - Added the existing "How to find the right answer fast" doc to the RR University's Browse-all-documents drawer (it had been missing from there since the doc was created).

May 19, 2026

- Corrected JD Edwards program references: R30822 (Frozen Cost Update) now replaces R30835 (Cost Simulation) across the analyzer and KB. R30835 is the cost-simulation preview step, not the frozen-update step. - Pattern 5.6 / 5.16 explanations tightened: R30837 (WIP Revaluation) writes both the F4111 "Standard Cost Change" row AND its GL counterpart, so the diagnosis now reads "R30837 partially fired -- GL side never wrote" instead of the older two-program sequencing-gap framing. - OV doc type (PO Receipt -- P4312) now correctly maps to AAI 4310. Was wrongly mapped to 4320 (voucher-match). Pattern 5.4's account-mismatch matrix labels OV docs correctly now. - Work-order Closed status is correctly described as UDC 00/SS (typically 90; customer-defined), not the sales / purchase-order 999. - DMAAI preload persists across browser restarts and reboots -- load once, sticks until you clear. - New Section 10 (JDE Status Code Reference -- WO / SO / PO) and Section 11 (Manufacturing Cost Programs Reference) in the Transaction Detail analyst guide. - New "Doc Type: OP / OV / PV" glossary entry in the PO Receipts reference doc.

May 18, 2026

- Three more Transaction Detail patterns use the cell-based JE-flow matrix in their "How to fix it" card: 5.6 Standard Cost Change, 5.16 Mfg Cost Mismatch, and 5.17 Voucher Variance (standard-cost branch). Matches the format Pattern 5.4 has been using. - JD Edwards cost method (UDC 40/CM) shows up on the "What happened" card for Patterns 5.6 and 5.15 too, plus any future / unmapped pattern. - Pattern 5.17's WHY text is shorter and more direct on both the std-cost and weighted-avg branches. - All bullet cards now have a blank line between bullets — the dense wall-of-text density is gone. - DMAAI preload persists across browser restarts and reboots. Load it once and it sticks; clear it explicitly via the Preload card when you want to switch customers. - New multi-customer F4095 preload script (Treatt and NACoal fixtures land alongside Acme).
