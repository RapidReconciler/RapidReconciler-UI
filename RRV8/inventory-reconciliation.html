Inventory Reconciliation reference
What each metric means, how the math works, and where to drill.

Variance components

Carry Forward
Prior period's unreconciled variance, rolled into the current period. If this is non‑zero, a previous close left a residual that hasn’t been booked. Click the Preview icon to see the prior period’s full breakdown.
GL Batches
F0911 batches hitting these inventory accounts that have not yet posted. The GL balance is short by this amount until they post.
End of Day
F4111 cardex records with no batch number — daily inventory movements that didn’t get bundled into a posted batch. Usually a JDE end‑of‑day job didn’t complete.
Transactions
Net F0911 vs F4111 timing/posting variance for the period. A positive raw amount reduces out‑of‑balance, so the math subtracts it.
Cardex
F41021 (item balance) vs F4111 (cardex) per‑item discrepancies. Items where the perpetual roll‑forward doesn’t tie — usually a cost or UM change that broke the roll.
Manual Journal Entries
Manual JEs booked directly to these inventory accounts in the period.
Unreconciled Variance
GL balance minus perpetual balance — the bottom‑line OOB. Equals the sum of the six components above (Transactions subtracted). When components sum cleanly to this total, every dollar is accounted for.

Common workflows

Investigating an out‑of‑balance period

  1. Pick the period from the header dropdown.
  2. Read the Variance Breakdown bottom‑up — the orange Unreconciled Variance is the target.
  3. For each non‑zero component, click Preview to see the contributing rows in the modal, or the download icon to export.
  4. Narrow the sidebar (Company / Object / BU / Subsidiary) when the breakdown is noisy — every widget recomputes in place.

Reading the Audit Report

  • Excel: one tab per company. Each tab is cover → Account Summary table → per‑account sections (Accounts Summary + the four sub‑sections + Perpetual Details).
  • PDF: same data, paginated. Every page carries Generated <ts> + Page X of Y.
  • Sidebar filters propagate to the export — deselected companies skip their tab.

Handling a Cardex variance

  1. Click Cardex Preview for per‑item rows with Reason (Amount or Quantity).
  2. Reason = Amount: needs a cost re‑roll. Reason = Quantity: needs an item‑balance fix.
  3. Narrow by Object or Subsidiary to scope the work list.

Handling End of Day

  1. Non‑zero means F4111 has un‑batched records — a JDE batch job didn’t bundle them.
  2. Preview shows per‑document detail; export to Excel and send to the JDE admin.
  3. Once the batch job runs, the next snapshot recapture clears the component.
