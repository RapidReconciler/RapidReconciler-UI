RapidReconciler Page won’t load

RapidReconciler won’t open at all — you see “This site can’t be reached” or a similar browser error instead of the login page. Run the quick check below first. If that doesn’t fix it, the button drafts a note to your IT so they can test from their side.

What you’re seeing

  • The login page never appears — the request fails before any RapidReconciler content loads.
  • Common phrasings: “the page won’t load,” “site can’t be reached,” “the URL is broken.”

Try this first

Step 1
Confirm the URL and hard-refresh

The RapidReconciler URL is rapidreconciler.getgsi.com. Run through these in order:

  • Check the address bar matches rapidreconciler.getgsi.com.
  • If you opened RR from a saved bookmark, update the bookmark to rapidreconciler.getgsi.com.
  • Hard-refresh the page — Shift+F5 on Windows or Cmd+Shift+R on Mac.

If the page still won’t load, draft an email to your IT — the email asks them to test from a different network and check filtering / DNS rules.
