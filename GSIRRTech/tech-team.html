Tech Team

GSI Tech owns the customer install and the underlying RapidReconciler product. Three roles share this work — this page covers what each one does and how work hands off between them. Reference docs for each role are listed inside the role cards below. (For the Helpdesk Tech role, see the Help Desk.)
